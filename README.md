# test_backend
## Installation

<ul dir="auto">
<li>Run <code>npm install</code></li>
</ul>

## Run
<ul dir="auto">
<li>Migrate <code>npm run migrate</code></li>
<li>Compile <code>npm run build</code></li>
<li>Start <code>npm run start</code></li>
</ul>
